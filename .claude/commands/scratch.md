# Scratchpad

Create a quick prototype or experiment in a temp file.

## Instructions

Based on the user's idea:

1. **Determine the language** from context or user input
2. **Create a scratch file** at `/tmp/scratch.<ext>` (e.g., `/tmp/scratch.py`, `/tmp/scratch.ts`)
3. **Write a minimal, runnable prototype** that demonstrates the concept
4. **Run it** to verify it works
5. **Show the output** and iterate if the user wants changes

The scratch file is disposable — optimize for speed and clarity, not production quality.
